# Prosperity Life & Financial Solutions — Static Site

Simple static website scaffold designed to generate inbound leads and booked appointments.

Quick start

1. Serve the folder locally (Python):

```bash
cd c:\Users\loret\Desktop\prosperity-website
python -m http.server 8000
```

2. Open http://localhost:8000 in a browser.

Forms

- The forms in `index.html` and `book.html` currently POST to `https://example.com/submit` (simulated). Replace that URL in `assets/js/main.js` with your form endpoint, or integrate a service like Formspree, HubSpot, or your CRM.
